배운 태그

큰 태그

  • !DOCTYPE HTML
    - 빠르게 변화하는 html의 버전을 지정한다는 것으로 이해하면 됨.

  • meta charset="urf-8"
    - 글자 깨짐 현상 방지

  • html
    - 시작과 끝에

  • head, body
    - head : 사이트 외의 부분. 즉, 이 창의 이름같은 위에, 범위가 큰 부분을 묶음.
    - body : 사이트 내의 부분. 즉, 이 창의 내용 부분을 묶음.

  • title
    - 웹 사이트의 이름을 정해줌.

작은 태그

  • a 태그
    - 속성 부여

  • strong 태그
    - 강조

  • h태그
    - 제목 표시

  • li태그
    - 리스트. ul, ol리스트 안에 포함됨.

  • br, p태그
    - br : 한 줄 띄워줌 사용에 따라 문단처럼 보이기도 함
    - p : 문단 표시
